The Restaurant

Fire is the Soul of Every Dish

Naar & Noor brings the high-altitude flavours of Nepal, Tibet and the greater Himalayan region to the heart of London. Our chefs — each trained in mountain kitchens across South Asia — craft dishes that honour centuries-old traditions while embracing the finest seasonal ingredients.

From hand-formed momos to flame-kissed sekuwa, from warming dal bhat to delicate Newari sweets, every plate tells a story of place and heritage.

This API is the backbone of our digital platform, letting guests explore our menu, book tables, place orders and share their experiences — seamlessly.

Architecture

How It's Built

A clean, modular architecture built for maintainability and performance.

🏗️

Clean Architecture

Four-layer separation: Domain, Application (CQRS + MediatR), Infrastructure (EF Core + PostgreSQL), and API (ASP.NET Core 8).

⚡

CQRS Pattern

Commands and Queries are cleanly separated. Every API action dispatches a typed MediatR request with full validation via FluentValidation.

🛡️

Security First

IP-based rate limiting (AspNetCoreRateLimit), security response headers, CORS policy enforcement, and structured error handling.

🗄️

PostgreSQL

Persistent data storage via Entity Framework Core with code-first migrations. Automatic database seeding on startup.

📊

Observability

Structured logging via Serilog (compact JSON format). Health check endpoint at /health for uptime monitoring.

📖

OpenAPI / Swagger

Full API documentation generated automatically at /api/docs. Try every endpoint directly in the browser.
